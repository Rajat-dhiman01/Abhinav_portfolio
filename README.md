## ✨ Features

- 🎭 **Premium Dark Luxury Design** - Custom dark color palette with gold accents, crafted to reflect stage presence and artistry
- 📱 **Fully Responsive** - Seamless experience across all screen sizes
- 🎬 **Smooth Animations** - Powered by Framer Motion and GSAP for cinematic transitions
- 🗓️ **Booking System** - Inquiry form for event and performance bookings
- 🖼️ **Gallery & Performances** - Showcase of photos and live performance highlights
- 📬 **Contact Page** - Direct communication channel for fans and event organizers
- ⚡ **Blazing Fast** - Vite-powered build for near-instant load times

## 🎨 Design System

The portfolio uses a custom **dark luxury** design language with the following color palette:

| Token | Description |
|-------|-------------|
| `primary` | Deep background - rich dark base |
| `secondary` | Elevated surface for cards/sections |
| `accent` | Gold highlight - calls to action |
| `accent-light` | Soft gold for subtle highlights |
| `cream` | Warm off-white for headings |
| `muted` | Subdued text for secondary content |

**Typography:**
- **Cinzel** - Display headings (classical, regal)
- **Cormorant Garamond** - Subheadings (elegant serif)
- **Inter / Manrope** - Body text (clean, readable)
